# Sprint document classification

Current hybrid execution authority is the unpadded sequence `SPRINT_0_...` through `SPRINT_14_...`, plus `SPRINT_4_1_...`, `SPRINT_6_1_...` and `SPRINT_12_1_...`.

Zero-padded reports such as `SPRINT_01_...` through `SPRINT_09_...`, and documents for the discontinued MedSpa/Seller execution track, are preserved historical migration evidence. Their completed labels do not make an old capability active or sellable in the hybrid REVORY.
